# Фитнес Секундомер

Тройной секундомер для фитнеса с расчетом калорий и круговыми тренировками. PWA-приложение для отслеживания круговых тренировок с возможностью расчета сожженных калорий.

## Особенности

- Три независимых секундомера для круговых тренировок
- Настройка количества кругов, времени упражнений и отдыха
- Автоматическое переключение между кругами
- Расчет сожженных калорий
- Адаптивный дизайн для мобильных устройств
- PWA-функциональность

## Технологии

- React
- Material UI
- JavaScript
